# Project 4 - Weather Dashboard

A React weather dashboard application built with Vite and the OpenWeather API.

## Features

* Search weather by city name
* Displays current temperature
* Uses OpenWeather API data
* Built with React and Vite
* Temperature displayed in imperial units

## Technologies Used

* React
* JavaScript
* Vite
* OpenWeather API
* CSS

## How to Run the Project

1. Clone the repository
2. Open the project folder
3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open the localhost link shown in the terminal
